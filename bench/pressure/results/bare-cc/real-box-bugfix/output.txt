All 115 tests pass. Here's a summary of the fix:

**Root cause:** In `Box.__contains__` (line 404), the code that traverses dotted key paths (`self[first_item]` and `children in it`) was in a `try/except/else` block where the `else` clause wasn't protected by the `except`. When an intermediate key was missing (e.g., `'grab'` in `'foo.grab.plum'`), `BoxKeyError` was raised from the `else` block and propagated up through `get()` instead of returning `False`.

**Fix:** Moved `self[first_item]` and `isinstance(it, Iterable) and children in it` from the `else` block into the `try` block, catching `(BoxError, KeyError, IndexError, TypeError)` so that missing keys at any depth return `False` instead of raising.
