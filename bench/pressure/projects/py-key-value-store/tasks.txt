Build a persistent key-value store library in Python. Requirements: KVStore class with get(key, default=None), set(key, value), delete(key), exists(key), keys(pattern=None) with glob matching, clear(). Values can be any JSON-serializable type. Storage backend: append-only log file with periodic compaction. When the log exceeds 1000 entries, compact by writing only latest values to a new file and swapping atomically. Must support concurrent readers with a single writer (use file locking). Include TTL support: set(key, value, ttl_seconds=None) where expired keys are cleaned up during compaction and not returned by get. Include comprehensive tests: basic CRUD, TTL expiry, compaction trigger, glob pattern matching, concurrent read/write safety with threading. Use only stdlib plus pytest.
