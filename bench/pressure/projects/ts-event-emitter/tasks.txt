Build a type-safe event emitter library in TypeScript. Requirements: TypedEmitter<EventMap> where EventMap is {eventName: payloadType}. Methods: on(event, handler), off(event, handler), once(event, handler), emit(event, payload), removeAllListeners(event?). Type safety: emitting 'click' with wrong payload type should be a compile error. Features: (1) wildcard listener via on('*', handler) that receives all events, (2) async handlers support — emitAsync(event, payload) returns Promise that resolves when all handlers complete, (3) maxListeners per event with warning (not error) when exceeded, (4) prepend(event, handler) to add handler at front of queue, (5) eventNames() returns list of events with active listeners. Include a Mediator class built on TypedEmitter that implements request/response: mediator.request(channel, data) returns Promise<response> by waiting for a registered handler. Tests: basic emit/listen, once fires only once, off removes correct handler, wildcard receives all events, async handlers, mediator request/response, type inference compile tests. Use typescript, jest, ts-jest.
