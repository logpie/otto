Build a thread-safe token bucket rate limiter library in Python. Requirements: TokenBucket class with consume(tokens=1) returning bool, configurable rate (tokens/sec) and burst capacity, sliding window support via SlidingWindowLimiter that tracks per-key limits, must be safe for concurrent access from multiple threads. Include comprehensive tests including a concurrency stress test that spawns 50 threads hitting the same limiter and verifies total consumed tokens never exceeds the expected limit within a time window. Use only stdlib (threading, time).
