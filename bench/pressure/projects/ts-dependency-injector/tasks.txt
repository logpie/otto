Build a lightweight dependency injection container in TypeScript. Requirements: Container class with register(token, factory, {scope}) and resolve(token). Scopes: 'singleton' (created once), 'transient' (new instance each resolve), 'scoped' (shared within a scope created by container.createScope()). Token can be string or Symbol. Factory receives the container for recursive resolution. Features: (1) circular dependency detection — throw clear error with the dependency chain, (2) auto-dispose: container.dispose() calls .dispose() on all singleton instances that have it, (3) child containers: createChild() inherits parent registrations but can override, (4) tagged resolution: register with tags, resolveAll(tag) returns all matching instances, (5) lazy resolution: container.lazy(token) returns a proxy that only resolves on first property access. Type safety: resolve<T>(token) returns T. Tests: singleton vs transient behavior, scoped lifecycle, circular detection with clear error message, child container override and fallback, dispose cleanup, tagged resolution, lazy proxy. Use typescript, jest, ts-jest.
