The debug-statements hook in pre-commit-hooks detects debug imports like pdb, ipdb, pdbr, etc., but it does not detect bpdb (a debugger that uses bpython as the interface). Users who use bpdb can accidentally leave "import bpdb" statements in their code without the hook catching it. Add bpdb to the set of detected debug statement modules in pre_commit_hooks/debug_statement_hook.py so that "import bpdb" and "from bpdb import ..." are flagged. Run tests with: python -m pytest
